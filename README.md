## ğŸš€ **Quick Start**

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/DevyRuxpin/rhode-island-fishing-guide.git

# Navigate to project directory
cd rhode-island-fishing-guide
```

```bash

# Start production server
```
